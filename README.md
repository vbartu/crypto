# Crypto

Implementation of some common cryptographic primitives.

* Block ciphers
    * AES 128
    * DES
* Modes of operation
    * ECB
    * CBC
    * CTR
* Hash functions
    * SHA224
    * SHA256
    * SHA384
    * SHA512
    * SHA512/224
    * SHA512/256
* Message authentication codes (MACs)
    * HMAC
